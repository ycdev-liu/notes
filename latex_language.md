


### 1️⃣ 基本用法（平方根）

```latex
\sqrt{x}
```

效果：
(\sqrt{x})

---

### 2️⃣ 带指数的根号（如立方根）

```latex
\sqrt[n]{x}
```

例如：

```latex
\sqrt[3]{x}
```

效果：
(\sqrt[3]{x})

---

### 3️⃣ 复杂表达式

```latex
\sqrt{a+b}
```

效果：
(\sqrt{a+b})

---

如果你是在写公式并想更直观理解这个结构，可以看这个标准形式👇
\sqrt[n]{x}

---



如果你说的是 **LaTeX 里常用的“希腊字母（很多人会误叫阿拉伯符号）”**，下面给你整理一份最常用的👇

---

## 🔤 小写希腊字母

```latex
\alpha   \beta   \gamma   \delta   \epsilon  
\zeta    \eta    \theta   \iota    \kappa  
\lambda  \mu     \nu      \xi      \pi  
\rho     \sigma  \tau     \phi     \chi  
\psi     \omega
```

效果示例：
(\alpha, \beta, \gamma, \theta, \lambda, \pi, \sigma, \phi, \omega)

---

## 🔠 大写希腊字母

```latex
\Gamma  \Delta  \Theta  \Lambda  \Xi  
\Pi     \Sigma  \Phi    \Psi     \Omega
```

效果：
(\Gamma, \Delta, \Theta, \Lambda, \Pi, \Sigma, \Omega)

---

## ⭐ 常用变体（很重要！）

```latex
\varepsilon   \varphi   \vartheta   \varpi   \varrho   \varsigma
```

效果：
(\varepsilon, \varphi, \vartheta, \varsigma)

👉 比如：

* `\epsilon` vs `\varepsilon`（两种 ε 形状）
* `\phi` vs `\varphi`

---

## 📐 常搭配的数学符号

顺便给你一些经常一起用的：

```latex
\infty     % ∞
\partial   % ∂
\nabla     % ∇
\pm        % ±
\cdot      % ·
\times     % ×
\leq       % ≤
\geq       % ≥
neq        % ≠
```

效果：
(\infty, \partial, \nabla, \pm, \times, \leq, \geq, \neq)

---

## 📌 小技巧

* 所有符号都必须在 **数学环境**中使用，比如：

```latex
$ \alpha + \beta = \gamma $
```

* 或者：

```latex
\[ \theta = \arctan(x) \]
```

---

在 **LaTeX** 里，“除法”有几种常用写法，取决于你想要的效果👇

---

## 1️⃣ 分数形式（最常用 ✅）

```latex
\frac{a}{b}
```

效果：
(\frac{a}{b})

👉 这是最标准、最推荐的写法（论文、作业都用这个）

---

## 2️⃣ 斜杠表示

```latex
a / b
```

效果：
(a / b)

👉 更像编程或简单表达

---

## 3️⃣ 除号符号（÷）

```latex
\div
```

效果：
(a \div b)

👉 小学/基础数学常见，但学术里用得少

---

## 📌 标准形式展示

\frac{a}{b}

---

## 💡 小建议

* 写正式公式 👉 用 `\frac`
* 快速打字 👉 用 `/`
* 教学或基础表达 👉 用 `\div`
